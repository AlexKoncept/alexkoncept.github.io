Ciné Secrets NEW

IA React UI/UX Cinéma

Ciné Secrets est une application web propulsée par l'IA, conçue pour révéler les détails cachés, les caméos et les "easter eggs" dissimulés dans les films et les séries. Dotée d'un moteur de recherche intelligent et d'un bouton "Surprends-moi", elle propose une exploration interactive et bilingue des œuvres cinématographiques, le tout dans un design moderne avec mode Sombre/Clair.
